# 4 Seasons Glazenwasser Website

Professionele raamschoonmaak en reiniging website voor 4 Seasons in Vlaanderen.

## Project Details

**Bedrijf:** 4 Seasons Glazenwasser  
**Locatie:** Vlaanderen, België  
**Contact:** +32 3 332 00 63 | info@4seasonscleaning.be  
**Website:** www.4seasonscleaning.be

## Diensten

1. **Glasbewassing** - Professionele reiniging van ramen en glazen oppervlakken
2. **Dakgoten Reinigen** - Onderhoud van dakgoten en goten
3. **Zonnepanelen** - Reiniging voor optimale efficiëntie
4. **Corniche** - Reiniging van corniche-systemen

## Features

- ✅ Responsive design (mobiel, tablet, desktop)
- ✅ Contact formulier met service selectie
- ✅ Portfolio galerij
- ✅ Over Ons pagina met brand story

- ✅ Smooth scrolling & animaties

## Deployment

1. Clone repository
2. Update image assets in `assets/images/`
3. Test alle links en formulieren
4. Deploy naar webserver
5. Configureer DNS records
